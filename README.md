# Smart EMI Calculator

A modern EMI calculator built using React, Tailwind CSS, and Recharts.

## Features
- EMI calculation
- Total interest calculation
- Total payment calculation
- Interactive pie chart
- Dark and light mode
- Repayment schedule
- PDF download
- Responsive design

## Tech Stack
- React
- Tailwind CSS
- Recharts
- jsPDF
